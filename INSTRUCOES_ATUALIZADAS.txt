=============================================
INSTRUÇÕES PARA CONFIGURAÇÃO LOCAL DO SISTEMA
=============================================

Siga estas instruções para configurar e executar o sistema em seu ambiente local.

PASSO 1: CONFIGURAR O XAMPP
--------------------------
1. Abra o XAMPP Control Panel
2. Inicie os serviços Apache e MySQL (clique nos botões "Start")
3. Verifique se os serviços estão rodando (devem ficar verdes)

PASSO 2: CONFIGURAR O BANCO DE DADOS
----------------------------------
1. Execute o arquivo "config_local.bat" clicando duas vezes nele
2. Este script vai:
   - Criar o banco de dados "laravel_beta2"
   - Configurar o arquivo .env
   - Gerar uma chave de aplicação
   - Limpar o cache

PASSO 3: EXECUTAR AS MIGRAÇÕES
----------------------------
1. Execute o arquivo "run_migrations.bat" clicando duas vezes nele
2. Este script vai criar todas as tabelas necessárias no banco de dados

PASSO 4: CRIAR USUÁRIO ADMINISTRADOR
---------------------------------
Você tem duas opções:

OPÇÃO A - Usando o phpMyAdmin:
1. Acesse http://localhost/phpmyadmin
2. Selecione o banco de dados "laravel_beta2"
3. Vá na aba SQL do phpMyAdmin
4. Cole o conteúdo do arquivo "criar_admin_apos_migracao.sql"
5. Clique em Executar

OPÇÃO B - Usando o script batch:
1. Execute o arquivo "create_admin.bat" clicando duas vezes nele

PASSO 5: INICIAR O SERVIDOR
-------------------------
1. Execute o arquivo "start_server.bat" clicando duas vezes nele
2. O servidor será iniciado na porta 8000
3. Acesse o sistema em seu navegador: http://localhost:8000

SOLUÇÃO DE PROBLEMAS
-------------------
Se encontrar problemas:

1. Verifique se o XAMPP está rodando (Apache e MySQL)
2. Verifique se o banco de dados foi criado corretamente
   - Abra o phpMyAdmin: http://localhost/phpmyadmin
   - Verifique se existe o banco "laravel_beta2"
3. Se necessário, execute as migrações manualmente:
   - Abra o prompt de comando na pasta do projeto
   - Execute: C:\xampp\php\php.exe artisan migrate --force
